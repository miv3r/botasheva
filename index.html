Формат интенсива

Интенсив “Хороший родитель” пройдет в формате zoom митингов. На протяжении 5-ти дней с 3 по 7 января мы будем встречаться каждый вечер на один час. У вас будет возможность отработать свои слабые места с психологом, а также получить ответы на свои вопросы.

Для кого

  • Для родителей детей любого возраста
  • Для будущих родителей
  • Для всех людей, которые не хотят травмировать своих детей. Хотят научится продуктивной коммуникации. Убрать насилие из отношений с детьми.
